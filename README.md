# Prodigy_InfoTech_Task4
# Sentiment Analysis of Social Media Data
Task 4 – Prodigy Infotech Data Science Internship
# Overview
This project analyzes Twitter data to understand public sentiment toward different topics and brands. Using NLP techniques, tweets were classified into Positive, Negative, Neutral, and Irrelevant categories.
# Objectives
*Clean and preprocess text data
*Perform sentiment analysis
*Visualize sentiment distribution
*Analyze topic-wise public opinion
# Tools & Libraries
*Python
*Panda
*NumPy
*Matplotlib
*Seaborn
*NLTK
*WordCloud
# Results
Negative sentiment is slightly dominant.
Positive sentiment is also high.
Neutral and Irrelevant tweets form a smaller portion.
Topic-wise analysis shows varying public reactions across brands.
Visualizations include pie charts, bar charts, and stacked bar charts to clearly represent sentiment trends.
# Conclusion
The project successfully identifies sentiment patterns in social media data and provides insights into public opinion using NLP and data visualization.
